## API
### getDefaultToolkit()
```java
public static synchronized Toolkit getDefaultToolkit()
```
- 获得默认的工具箱。工具箱包含有关GUI环境的信息
```java
Toolkit.getDefaultToolkit().beep();
```
### beep()
```java
public abstract void beep();
```
- 发出一声铃响
