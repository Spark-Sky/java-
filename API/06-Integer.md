## API
### parseInt
```java
public static int parseInt(String s) throws NumberFormatException 
```
- `s`要转换的字符串
```java
Integer a = Integer.parseInt("123");
```

### parseInt
```java
public static int compare(int x, int y)
```
- 比较`x`和`y`两个整型
- 如果`x<y`返回-1
- 如果`x>y`返回1
- 相等返回0
```java
Integer a = Integer.parseInt("123");
```
